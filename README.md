# terminal-chat-app

A real-time terminal-based chat application built with Ink and RelayX.

## Setup

1. Navigate to the app directory:
```bash
cd terminal-chat-app
```

2. Install dependencies:
```bash
npm install
```

3. Build the application:
```bash
npm run build
```

4. Link the app globally:
```bash
npm link
```

## Run

After setup, run the application from anywhere:

```bash
terminal-chat-app
```

## Development

To run in development mode with automatic rebuilding on file changes:

```bash
npm run dev
```

Then in another terminal, run:

```bash
terminal-chat-app
```
